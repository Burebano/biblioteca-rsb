¿Qué es la biblioteca de componentes RSB Library?
Es un conjunto de componentes, servicios, pipes y directivas reutilizables para tus proyectos.
¿Cómo puedo instalar esta maravillosa biblioteca?
Ejecutando el siguiente comando: 
npm install rsb-library
¿Qué debo añadir en mi angular.json?
Añadiremos lo siguiente en assets: 
{ "glob": "**/*", "input": "node_modules/rsb-library/resources/img", "output": "resources/img/"}
Añadiremos lo siguiente en styles: 
"node_modules/bootstrap/dist/css/bootstrap.min.css",
"node_modules/font-awesome/css/font-awesome.min.css",
"node_modules/rsb-library/resources/scss/rsb-styles.scss",
"node_modules/rsb-library/resources/themes/theme-default.scss",
"node_modules/rsb-library/resources/themes/theme-dark.scss",
"node_modules/rsb-library/resources/scss/font.scss"
¿Qué debo añadir en mi app.module? 
import { BrowserAnimationsModule } from '@angular/platform-browser/animations';
import { RsbLibraryModule } from 'rsb-library';
@NgModule({
  imports: [
    BrowserModule,
    BrowserAnimationsModule,
    RsbLibraryModule
  ],
  bootstrap: [AppComponent]
})
